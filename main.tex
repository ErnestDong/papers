  公司的违约实施点为企业1年以下短期债务的价值加上未清偿长期债务账面价值的一半
\item
  贷款期限一年
\item
  当公司发生违约时，违约回收率为0，即全部贷款均不能回收。
\item
  贷款利率为包括流动性溢价和信用溢价，分别用无风险利率和预期违约率\$\times\$100表示，即贷款利率
  = 无风险利率 + 预期违约率 \(\times\) 100
\end{itemize}
